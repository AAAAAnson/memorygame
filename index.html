How to Play

Game Rules

Face 20 challenging levels of increasing difficulty. Each level presents a number that you must memorize within 5 seconds.

Challenge Progression

Start with a single digit and progress to 20-digit numbers. Each level tests your memory capacity further.

Time Pressure

You have 5 seconds to memorize each number before it disappears. Think fast and stay focused!

Scoring System

Level Number Length Points
Level 1 1 digit 5 points
Level 2 2 digits 10 points
Level 20 20 digits 100 points

Maximum possible score: 1050 points. Score above 840 points is considered excellent!
